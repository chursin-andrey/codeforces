A. Следующий раунд

ограничение по времени на тест: 3 seconds
ограничение по памяти на тест: 256 megabytes

«Участник выходит в следующий раунд соревнований, если он набрал не меньше баллов, чем участник, занявший k-е место, при условии, что набранное им число баллов строго положительно...» — выдержка из правил соревнований.Всего в соревновании было n участников (n ≥ k), и вам уже известны их баллы. Определите, сколько участников выйдет в следующий раунд.

Входные данные: В первой строке входных данных записаны два целых числа n и k (1 ≤ k ≤ n ≤ 50), разделенных единичным пробелом.
Выходные данные: Выведите одно целое число — искомое количество участников, в соответствии с правилами.

Примеры тестов:

Пример входного файла (input.txt)Пример выходного файла (output.txt)
4 2
0 0 0 0
0
8 5
10 9 8 7 7 7 5 5
6
