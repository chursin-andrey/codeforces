A. Почти простые числа

ограничение по времени на тест: 2 seconds
ограничение по памяти на тест: 256 megabytes

Число называется почти простым, если оно имеет ровно два различных простых делителя. Например, числа 6, 18, 24 являются почти простыми, а 4, 8, 9, 42 — не являются. Найдите количество почти простых чисел от 1 до n включительно.

Входные данные: В первой строке входного файла записано число n (1 ≤ n ≤ 3000).
Выходные данные: Выведите количество почти простых чисел от 1 до n включительно.

Примеры тестов:

Пример входного файла (input.txt)Пример выходного файла (output.txt)
102
218
